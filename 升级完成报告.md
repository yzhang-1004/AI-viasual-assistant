## 🔧 技术改进

### API 调用优化：
| 接口 | 旧版 | 新版 |
|------|------|------|
| ASR | `input: { audio }` | `input: { file_urls: [] }` |
| TTS | `input: text` | `input: { text }` |
| LLM | 完整 options 对象 | 简化参数 |
| VLM | 完整 options 对象 | 简化参数 |

### 响应格式统一：
```typescript
// 所有 API 统一返回格式
{
  success: boolean,
  response?: string,    // 对话接口
  text?: string,        // ASR 接口
  error?: string
}
```
