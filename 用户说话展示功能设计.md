# 用户说话文字展示功能设计

## 现状

摄像头在左上角，下方只有一个灰色 "你" 标签。用户说话时没有视觉反馈。

```
┌──────────┐
│  摄像头  │
└──────────┘
    你          ← 只有这个
```

## 目标

用户说话时，识别出的文字实时显示在摄像头下方，说完消失。

```
┌──────────┐
│  摄像头  │
└──────────┘
我今天心情不太好...    ← 用户说的话（淡入淡出）
```

## 改动点（3 处）

### 1. `appStore.ts` — 加一个状态
```
userText: string          // 用户当前说的话
setUserText(text)         // 更新
```

### 2. `useConversation.ts` — 识别到文字时写入
在 `recognition.onresult` 里，拿到 interim/final 后：
```
setUserText(interim + final)    // 实时更新
```
AI 开始回复时清空：
```
setUserText('')
```

### 3. `UserCameraWindow.tsx` — 在摄像头下方渲染
在 "你" 标签下方加一行文字：
```
{userText && (
  <div className="...动画...">
    {userText}
  </div>
)}
```
样式：浅色半透明、小字、最大两行、带淡入动画。

## 交互规则

| 状态 | 显示 |
|------|------|
| 用户正在说话 | 实时显示识别文字 |
| AI 回复中 | 清空 |
| 静默 | 不显示 |
